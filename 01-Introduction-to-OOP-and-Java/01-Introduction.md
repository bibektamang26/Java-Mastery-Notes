# 1.1 Introduction

Java is a **high-level, object-oriented, platform-independent programming language** developed by **Sun Microsystems** in **1995**. It was created by **James Gosling** and his team with the goal of developing a language that was simple, secure, reliable, and capable of running on different types of computers without modification.

The philosophy behind Java is **Write Once, Run Anywhere (WORA)**. This means that a Java program only needs to be written and compiled once. The compiled program (called **bytecode**) can then run on any operating system, such as Windows, Linux, or macOS, as long as that system has a **Java Virtual Machine (JVM)** installed.

---

## Why is Java called a High-Level Language?

A **high-level programming language** is a language that is easy for humans to read, write, and understand. Unlike machine language, which consists of binary numbers (0s and 1s), Java uses simple English-like keywords such as `if`, `while`, `class`, and `public`.

Because of its readable syntax, programmers can develop applications more quickly and maintain them more easily.

**Example:**

```java
System.out.println("Hello, World!");
```

The above statement is easy to understand, unlike machine code.

---

## Why is Java called Object-Oriented?

Java follows the **Object-Oriented Programming (OOP)** paradigm. In OOP, a program is built using **classes** and **objects** instead of only functions and procedures.

Objects represent real-world entities. For example, a **Student**, **Car**, or **Bank Account** can all be represented as objects in Java.

Using OOP makes programs:

- Easier to understand
- Easier to maintain
- Reusable
- More secure
- Suitable for developing large applications

The four main principles of OOP are:

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

These concepts will be discussed in detail in the next topic.

---

## Why is Java Platform Independent?

Normally, programs written in languages like C or C++ are compiled directly into machine code, which is specific to one operating system.

Java works differently.

When a Java program is compiled, it is converted into **bytecode**, not machine code. This bytecode is then executed by the **Java Virtual Machine (JVM)**.

Since JVMs are available for different operating systems, the same Java program can run on any platform without changing the source code.

This is why Java is known as a **platform-independent language**.

```
Java Source Code (.java)
          │
          ▼
      Java Compiler
          │
          ▼
      Bytecode (.class)
          │
          ▼
     Java Virtual Machine (JVM)
          │
          ▼
   Windows / Linux / macOS
```

# History of Java

Java was developed by **James Gosling**, **Mike Sheridan**, and **Patrick Naughton** at **Sun Microsystems**. Development began in **1991** as part of the **Green Project**, whose goal was to create software for consumer electronic devices such as televisions and set-top boxes.

Initially, the language was named **Oak**, inspired by the oak tree outside James Gosling's office. However, the name "Oak" was already registered as a trademark by another company, so it had to be changed.

In **1995**, the language was officially renamed **Java** and released to the public. The name "Java" was inspired by Java coffee, reflecting the team's informal brainstorming sessions over coffee.

One of Java's biggest innovations was its ability to run the same program on different operating systems without recompilation. This concept became known as **Write Once, Run Anywhere (WORA)** and remains one of Java's defining features.

In **2010**, **Oracle Corporation** acquired Sun Microsystems and became the owner and primary developer of Java. Today, Oracle continues to release new Java versions with improved features, performance, and security.

## Why is Java Popular?

Java is one of the most popular programming languages because it provides many useful features:

- **Simple** – Easy to learn and use.
- **Object-Oriented** – Organizes programs into classes and objects.
- **Platform Independent** – Runs on different operating systems.
- **Secure** – Protects programs from many common security issues.
- **Robust** – Handles errors efficiently and manages memory automatically.
- **Portable** – Programs can run on different hardware platforms.
- **Multithreaded** – Can perform multiple tasks simultaneously.
- **High Performance** – Uses Just-In-Time (JIT) compilation for better execution speed.

These features make Java suitable for both small and large-scale software development.
